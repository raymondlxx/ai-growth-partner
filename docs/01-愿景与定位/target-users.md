## 核心用户详细画像

### 用户群1：职场转型者

**人口统计**
- 年龄：28-40岁
- 职业：互联网/金融/传统行业从业者
- 学历：本科以上
- 收入：15-40万/年

**核心痛点**
- 现有技能面临淘汰焦虑
- 不知道如何系统转型
- 工作繁忙，时间碎片化
- 家庭压力，无法all-in学习

**决策特点**
- 愿意为"确定性"付费
- 注重实际效果和案例
- 需要快速看到进展

**典型场景**
```
"我在传统行业干了5年，感觉要被淘汰了。想转行AI相关，但不知道从哪里开始，太多选择反而不知道怎么选。"
```

### 用户群2：宝妈群体

**人口统计**
- 年龄：25-35岁
- 状态：孩子0-6岁
- 背景：曾有职业经历，大专以上
- 时间：碎片化（孩子睡眠时）

**核心痛点**
- 与职场脱节的焦虑
- 想证明自己价值
- 时间极度碎片
- 需要兼顾家庭

**决策特点**
- 倾向社群归属感
- 需要正向激励和陪伴
- 容易被"宝妈逆袭"故事激励

**典型场景**
```
"孩子上幼儿园后，我想重新工作，但gap了3年不知道能做什么。想学点实用的技能，最好能兼顾带娃时间。"
```

### 用户群3：应届/职场新人

**人口统计**
- 年龄：20-25岁
- 状态：学生或初入职场
- 背景：本科在读或刚毕业
- 时间：相对充裕

**核心痛点**
- 职业方向不明确
- 技能与市场需求脱节
- 缺乏实战经验
- 信息过载，选择困难

**决策特点**
- 愿意尝试，容错率高
- 受KOL影响大
- 喜欢社交和互动

**典型场景**
```
"秋招了，我不知道该投什么岗位。看很多人在学AI，我也想学，但不确定学了能做什么。"
```

## 用户分层

### 第一层：天使用户（种子期）

**特征**
- 有强烈痛点和改变意愿
- 愿意投入时间参与测试
- 愿意提供反馈

**获取方式**
- 朋友圈/社群精准招募
- KOL推荐体验
- 垂直社群合作

**数量目标**: 100-200人

### 第二层：核心用户（MVP期）

**特征**
- 有明确的学习目标
- 愿意为优质内容付费
- 有社交传播意愿

**获取方式**
- 社交媒体投放
- 内容营销
- 口碑传播

**数量目标**: 5000-10000人

### 第三层：生态用户（扩展期）

**特征**
- 各类需求的用户
- 愿意参与内容共创
- 可以是内容的消费者也可以是生产者

**获取方式**
- 平台品牌效应
- SEO/应用商店
- 企业合作

## 用户研究方法

| 阶段 | 方法 | 目的 |
|------|------|------|
| 规划期 | 深度访谈10-20人 | 验证痛点，收集需求 |
| MVP期 | 问卷调研+行为分析 | 优化功能，验证价值 |
| 运营期 | A/B测试+用户反馈 | 持续迭代 |
