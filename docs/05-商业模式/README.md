## Unit Economics

### 关键指标

```
┌─────────────────────────────────────────────────────────────┐
│                    单位经济模型                             │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  LTV (用户生命周期价值)                                      │
│  ├── 平均付费用户生命周期：18个月                           │
│  ├── 平均月收入：40元                                       │
│  └── LTV = 40 × 18 = 720元                                 │
│                                                              │
│  CAC (用户获取成本)                                         │
│  ├── 付费投放：150元                                        │
│  ├── 口碑推荐：50元                                         │
│  └── 加权平均：100元                                       │
│                                                              │
│  LTV/CAC = 720 / 100 = 7.2  ✓ (目标 > 3)                    │
│                                                              │
│  回本周期：约 3-4 个月                                      │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```
