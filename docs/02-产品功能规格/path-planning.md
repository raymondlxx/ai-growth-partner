# AI路径规划引擎

## 功能概述

基于"元路径"概念，将技能路径与人生阶段挂钩，为用户生成个性化的成长路径，并支持动态调整。

## 核心概念

### 元路径（Meta-Path）

将用户的**人生阶段**与**技能组合**关联，形成"可复用、可调整"的路径模板。

```
元路径 = 人生阶段切片 × 技能组合 × 时间节奏

示例1：宝妈阶段
├── 阶段1: 育儿期（0-2年）→ 学习: 时间管理、情绪调节、亲子技能
├── 阶段2: 副业探索期 → 学习: 技能变现、电商/内容创作
└── 阶段3: 重返职场期 → 学习: 行业技能、面试策略

示例2：中年职场人
├── 阶段1: 技能保鲜期 → 深化: AI工具、数据分析
├── 阶段2: 职业转型期 → 切换: 新领域基础技能
└── 阶段3: 第二曲线期 → 构建: 自由职业/创业能力
```

### 技能组合推荐

基于行业趋势和用户画像，推荐"AI+X"复合技能：

| 方向 | 推荐技能组合 | 变现路径 |
|------|-------------|----------|
| AI+教育 | AI教学设计+内容创作 | 在线辅导/课程销售 |
| AI+健康 | AI健康管理+心理知识 | 健康咨询/生活方式博主 |
| AI+商业 | AI数据分析+商业思维 | 商业顾问/数据分析师 |
| AI+创意 | AI绘图+设计基础 | 设计师/创意顾问 |

## 功能模块

### 1. 路径生成器

**输入**：
- 用户画像（基础属性+目标）
- 当前技能评估
- 生活阶段信息

**输出**：
- 阶段化学习路径（每个阶段有明确目标、任务、里程碑）
- 预计时间周期
- 推荐资源列表

**算法**：
```
输入 → 人生阶段匹配 → 技能缺口分析 → 路径规划 → 专家校准 → 输出
```

### 2. 路径动态调整

| 触发事件 | 调整策略 |
|----------|----------|
| 用户反馈太难 | 降低难度，增加前置任务 |
| 用户反馈太简单 | 跳过已掌握内容，压缩周期 |
| 生活事件变化 | 重排优先级，调整阶段时长 |
| 新技能趋势 | 推荐补充学习 |

### 3. 行业专家校准

- 各领域专家对AI生成的路径进行权重调整
- 确保专业性与前瞻性
- 专家库维护：引入机制、更新机制

## 路径结构

```
学习路径（以"AI产品经理"为例）
├── 前置评估（2天）
│   ├── 技能摸底测试
│   └── 了解学习背景
│
├── 基础构建阶段（4周）
│   ├── 第1周: AI基础概念
│   ├── 第2周: 产品管理基础
│   ├── 第3周: AI行业认知
│   └── 第4周: 实战项目准备
│
├── 技能深化阶段（6周）
│   ├── 第5-6周: 用户研究+需求分析
│   ├── 第7-8周: AI模型理解
│   └── 第9-10周: 项目实战
│
├── 作品集阶段（4周）
│   ├── 构建AI产品设计文档
│   ├── 模拟面试准备
│   └── 简历+面试作品集
│
└── 成果验收
    ├── 模拟面试评估
    └── 获得认证/推荐信
```

## 验收标准

- [ ] 能根据用户画像生成初始路径（<30秒）
- [ ] 支持至少5种元路径模板
- [ ] 用户可以手动调整路径优先级
- [ ] 路径调整后24小时内生效
- [ ] 专家可审核和调整路径权重

## 技术实现要点

- 路径生成采用规则引擎+AI模型混合
- 知识图谱构建：技能-行业-职业关系
- 强化学习用于路径动态优化
- 引入专家反馈回路，持续优化模型
