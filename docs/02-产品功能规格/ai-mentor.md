# AI导师模块

## 功能概述

AI导师是"成长伴侣"的核心交互入口，提供个性化陪伴、即时答疑、情感支持。

## 角色定位

```
┌─────────────────────────────────────────────────────────────┐
│                     AI导师                                  │
├─────────────────────────────────────────────────────────────┤
│  角色1: 学习教练                                          │
│  └── 规划指导、进度跟踪、技巧传授                          │
│                                                             │
│  角色2: 情感陪伴者                                         │
│  └── 倾听、鼓励、疏导、正向反馈                            │
│                                                             │
│  角色3: 答疑助手                                           │
│  └── 即时问答、错误诊断、资源推荐                          │
│                                                             │
│  角色4: 成长伙伴                                          │
│  └── 分享成功案例、庆祝里程碑、共同成长                    │
└─────────────────────────────────────────────────────────────┘
```

## 核心功能

### 1. 对话式交互

**交互形式**：

| 形式 | 触发场景 | 示例 |
|------|----------|------|
| 主动问候 | 每日首次打开、任务完成 | "早上好！今天计划学习第三章，继续加油~" |
| 被动响应 | 用户主动提问 | "这个概念有点抽象，能举个例子吗？" |
| 主动干预 | 检测到用户困难/情绪 | "注意到你卡在这个任务很久了，需要帮忙吗？" |
| 成长播报 | 达成里程碑 | "恭喜完成第一阶段！你的Python水平提升了30%" |

### 2. 情感计算

**能力**：

- 语音情感识别（语调、语速）
- 文本情感分析（用词、句式）
- 行为情感推断（使用频率、活跃度）

**响应策略**：

```
情绪识别 → 情绪类型 → 响应策略

正面情绪（兴奋/满足）
└── 强化正向反馈 → 鼓励继续

负面情绪（沮丧/焦虑）
└── 降低难度/提供安慰 → 调整任务/情感支持

中性情绪（困惑/疲惫）
└── 提供帮助/休息建议 → 分步指导/合理安排
```

### 3. 学习指导

**能力范围**：

```
AI导师能提供的指导：
├── 学习规划
│   ├── 根据进度调整计划
│   ├── 推荐下一阶段内容
│   └── 解释学习路径的价值
│
├── 技能讲解
│   ├── 概念解释（比喻、例子）
│   ├── 技巧分享（经验之谈）
│   └── 资源推荐（教程/文章）
│
├── 实战辅导
│   ├── 代码Review/优化建议
│   ├── 方案评审
│   └── 模拟面试
│
└── 学习方法
    ├── 记忆技巧
    ├── 时间管理
    └── 克服拖延
```

### 4. 个性化定制

**导师形象**：

```
用户可选择导师风格：
├── 风格：严肃/温和/活泼/幽默
├── 领域：通用/职场/妈妈/学生
└── 人设：导师/学长/朋友/教练
```

**交互偏好**：

- 消息频率（每日几次、只在需要时）
- 详细程度（简短/中等/详细）
- 语言风格（专业/口语化）

## 会话设计

### 典型会话场景

#### 场景1：用户卡在任务中

```
用户：卡住了，不知道怎么继续
AI导师：[识别到卡顿]
    → 先肯定："这个确实有点难，你已经走到这里很棒了"
    → 提供线索："提示：看看第二章的第三小结，关于XX原理"
    → 询问是否需要更多帮助
```

#### 场景2：用户情绪低落

```
用户：好累，不想学了
AI导师：[识别到消极情绪]
    → 情感共鸣："能理解，工作/带娃已经很累了"
    → 降低预期："今天少学一点也没关系，保持学习习惯更重要"
    → 提供选择："要不今天就复习一下之前的内容，轻松一点？"
```

#### 场景3：用户达成里程碑

```
用户：（完成了一个大任务）
AI导师：[检测到完成]
    → 真诚庆祝："太棒了！这个项目完成得很漂亮"
    → 总结成长："看，你现在已经掌握了XX能力"
    → 展望下一步："准备好进入下一关了吗？"
```

## 技术实现要点

### 对话系统架构

```
用户输入 → 意图识别 → 情绪分析 → 对话策略 → 响应生成 → 输出

意图识别：识别用户想要什么（提问/求助/倾诉/分享）
情绪分析：判断用户情绪状态
对话策略：根据意图+情绪+用户画像，决定如何响应
响应生成：生成合适的回复内容
```

### 知识库

- 平台知识库：课程内容、任务说明、常见问题
- 通用知识库：学习方法、职业发展、生活技巧
- 个人知识库：用户历史记录、偏好设置、成长轨迹

### 隐私与边界

- 明确告知用户AI的能力边界
- 不提供医疗/法律等专业建议
- 用户数据加密，不用于其他目的
- 提供"AI关闭"选项

## 验收标准

- [ ] AI导师能识别用户意图（准确率>85%）
- [ ] 能进行至少10轮连续对话
- [ ] 情绪识别准确率>70%
- [ ] 响应时间<3秒
- [ ] 用户对AI导师的满意度评分（5分制）>4分

## 与其他模块的关系

```
AI导师 ← 用户画像系统（了解用户）
AI导师 ← 路径规划引擎（推荐下一步）
AI导师 ← 任务系统（追踪进度、触发辅导）
AI导师 → 用户（核心交互入口）
```
