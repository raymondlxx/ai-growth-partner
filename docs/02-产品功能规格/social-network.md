# 社交网络

## 功能概述

打造"成长者共同体"，通过社群、导师匹配、挑战赛等功能，提升用户粘性和传播力。

## 核心功能

### 1. 兴趣小组

**小组类型**：

| 类型 | 示例 | 特点 |
|------|------|------|
| 技能型 | "Python进阶圈"、"AI产品经理群" | 同技能互相学习 |
| 阶段型 | "宝妈副业联盟"、"职场转型互助" | 同阶段互相支持 |
| 行业型 | "互联网人"、"传统行业转型" | 行业信息共享 |
| 兴趣型 | "读书会"、"跑步打卡" | 共同兴趣 |

**小组功能**：

```
小组内功能：
├── 每日话题讨论
├── 作业/项目打卡
├── 经验分享
├── 专家答疑（定期 AMA）
└── 线下活动组织
```

### 2. 导师匹配

**导师来源**：

- 行业专家（付费咨询）
- 平台认证的资深用户
- 往期优秀学员

**匹配方式**：

```
用户需求 → 导师筛选 → 匹配推荐 → 预约咨询

匹配维度：
├── 领域匹配：同技能/行业
├── 阶段匹配：过来人经验
├── 风格匹配：沟通方式兼容
└── 价格匹配：预算范围内
```

**咨询形式**：

| 形式 | 时长 | 定价参考 |
|------|------|----------|
| 图文咨询 | 不限 | 50-200元/次 |
| 语音咨询 | 30分钟 | 200-500元/次 |
| 视频咨询 | 60分钟 | 500-1000元/次 |
| 长期陪跑 | 1-3个月 | 2000-5000元/月 |

### 3. 成长排名与挑战赛

**排行榜类型**：

```
┌─────────────────────────────────┐
│      成长排行榜                  │
├─────────────────────────────────┤
│  本周学习时长 TOP 10            │
│  任务完成率 TOP 10              │
│  技能提升速度 TOP 10            │
│  内容贡献 TOP 10                │
└─────────────────────────────────┘
```

**挑战赛**：

| 挑战类型 | 周期 | 示例 | 奖励 |
|----------|------|------|------|
| 技能挑战赛 | 1-2周 | AI绘画大赛、编程马拉松 | 奖金/徽章/认证 |
| 习惯打卡赛 | 30天 | 每日单词、每日一题 | 连续徽章 |
| 学习闯关赛 | 1月 | 一个月掌握Python | 实物奖励/面试机会 |
| 创意征集赛 | 不限 | 最佳学习笔记 | 现金奖励 |

### 4. 成长故事

**内容形式**：

- 用户投稿：学习历程、转型故事、成长心得
- 平台整理：采访、撰稿、视频制作
- 传播渠道：公众号、短视频、用户社群

**激励**：

- 被选中的故事获得现金奖励
- 入选"榜样说"专栏
- 获得更多曝光和关注

## 社交关系

```
用户关系类型：
├── 关注：单向关注任何用户
├── 好友：双向关系，可私信
├── 同学：同一路径/小组的伙伴
├── 导师：咨询关系转化
└── 粉丝：关注用户的粉丝
```

## 验收标准

- [ ] 社交小组功能完整（创建/加入/发言）
- [ ] 导师匹配系统能正常运行
- [ ] 排行榜每周更新
- [ ] 每季度至少举办1次挑战赛
- [ ] 社交功能对DAU贡献>20%

## 技术实现要点

- 私信系统：敏感词过滤、频率限制
- 小组：权限管理、内容审核
- 实时通知：推送系统
- 推荐系统：好友推荐、小组推荐
