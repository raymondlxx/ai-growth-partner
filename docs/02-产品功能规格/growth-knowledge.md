# 成长智库

## 功能概述

聚合用户的学习沉淀，构建个人知识库，同时开放内容共创，实现"学-沉淀-分享-变现"闭环。

## 核心能力

### 1. 个人知识库

**自动沉淀内容**：

```
用户学习过程中自动积累：
├── 学习笔记（AI辅助整理）
├── 完成的项目成果
├── 获得的证书/徽章
├── 学习轨迹记录
└── 与AI导师的对话精华
```

**知识组织**：

```
个人知识库结构：
├── 技能维度：按技能分类整理
│   ├── Python
│   │   ├── 基础知识
│   │   ├── 实战项目
│   │   └── 学习心得
│   └── 数据分析
│       └── ...
│
├── 时间维度：按学习时间线组织
│   ├── 2026年Q1
│   └── 2026年Q2
│
└── 成果维度：按产出类型分类
    ├── 证书
    ├── 项目
    └── 文章
```

### 2. 内容共创平台

**用户贡献内容**：

| 内容类型 | 审核机制 | 收益模式 |
|----------|----------|----------|
| 教程/攻略 | AI预审+人工复审 | 付费课程分成 |
| 案例分析 | AI预审+人工复审 | 付费课程分成 |
| 问答精选 | AI自动审核 | 积分奖励 |
| 实战经验 | AI预审+人工复审 | 付费课程分成 |

**审核流程**：

```
用户提交内容
    ↓
AI质量评分（完整性/准确性/原创性）
    ↓
评分≥80% → 直接上线
评分60-80% → 人工复审
评分<60% → 反馈修改意见
```

### 3. 技能变现通道

**变现路径**：

```
┌─────────────────────────────────────────────────────┐
│              技能变现闭环                            │
├─────────────────────────────────────────────────────┤
│                                                     │
│   学习成果 ──→ 个人品牌 ──→ 变现机会                 │
│       │           │           │                    │
│       │           │           ▼                    │
│       │           │      ┌────────┐               │
│       │           │      │兼职任务│               │
│       │           │      └────────┘               │
│       │           │           │                    │
│       ▼           ▼           ▼                    │
│   ┌─────────────────────────────┐                  │
│   │     技能认证 + 作品集        │                  │
│   │  → 接入自由职业平台         │                  │
│   │  → 平台内推荐兼职/就业      │                  │
│   └─────────────────────────────┘                  │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**对接平台**：

| 平台类型 | 对接方式 | 可变现技能 |
|----------|----------|------------|
| 自由职业平台 | API接入 | 设计/编程/写作 |
| 招聘平台 | 合作推荐 | 全职/兼职 |
| 内容平台 | 内容分发 | 知识付费 |

## 功能模块

### 1. 知识沉淀

- 自动整理用户的笔记和项目
- AI辅助优化知识结构
- 支持多格式导出（PDF/Notion/Markdown）

### 2. 内容贡献

- 贡献编辑器（支持Markdown）
- 贡献审核状态追踪
- 贡献者排名和积分

### 3. 变现中心

- 技能认证申请
- 作品集展示页
- 变现机会匹配

## 验收标准

- [ ] 用户可查看自己的知识沉淀
- [ ] 内容贡献审核流程完整（提交→审核→上线/反馈）
- [ ] 至少对接3个外部变现平台
- [ ] 创作者分成比例透明（平台:创作者 = 3:7）
