# 风险管理

## 风险矩阵

```
                        影响程度
              低         中         高
概率    ┌────────┬────────┬────────┐
  高    │ 信息过载│ 用户流失│ 技术风险│
        │ 内容质量│ AI成本  │ 团队建设│
        ├────────┼────────┼────────┤
  中    │ 竞争加剧│ 留存率低│ 合规风险│
        │ 变现难  │         │ 资金断裂│
        ├────────┼────────┼────────┤
  低    │         │ 宏观环境│         │
        │         │        │         │
        └────────┴────────┴────────┘
```

## 重点风险详情

### 1. 技术风险

#### AI能力不及预期

| 项目 | 说明 |
|------|------|
| 风险 | AI路径规划、AI导师效果达不到预期，影响用户体验 |
| 影响 | 高 |
| 概率 | 中 |
| 应对 | 规则引擎作为兜底；持续收集用户反馈迭代；与AI专家合作 |
| 指标 | AI导师满意度 > 4分，路径完成率 > 50% |

#### 技术债务积累

| 项目 | 说明 |
|------|------|
| 风险 | 快速迭代导致技术债务，影响后续扩展 |
| 影响 | 中 |
| 概率 | 高 |
| 应对 | MVP阶段控制技术债务；定期重构专项；代码审查机制 |
| 指标 | 技术债务专项不超过20%开发时间 |

#### 数据安全/隐私

| 项目 | 说明 |
|------|------|
| 风险 | 用户数据泄露、隐私合规问题 |
| 影响 | 高 |
| 概率 | 中 |
| 应对 | 数据加密存储；隐私合规咨询；最小化数据收集 |
| 指标 | 无重大数据安全事件 |

### 2. 产品风险

#### 用户留存率低

| 项目 | 说明 |
|------|------|
| 风险 | 用户完成初始体验后不再回来 |
| 影响 | 高 |
| 概率 | 中 |
| 应对 | 快速迭代；社交功能加入；情感陪伴强化；A/B测试 |
| 指标 | 次月留存 > 30% (MVP), > 40% (扩展) |

#### 任务完成率低

| 项目 | 说明 |
|------|------|
| 风险 | 用户无法完成学习任务，挫败感强 |
| 影响 | 中 |
| 概率 | 中 |
| 应对 | 难度自适应调整；AI辅导强化；新手引导优化 |
| 指标 | 任务完成率 > 50% |

#### 内容质量差

| 项目 | 说明 |
|------|------|
| 风险 | 课程内容、用户贡献内容质量参差不齐 |
| 影响 | 中 |
| 概率 | 高 |
| 应对 | AI+人工双重审核；用户评分机制；末尾淘汰 |
| 指标 | 内容评分 > 4分 (满分5分) |

### 3. 商业风险

#### AI成本失控

| 项目 | 说明 |
|------|------|
| 风险 | LLM调用量大，成本超过收入 |
| 影响 | 中 |
| 概率 | 高 |
| 应对 | 缓存优化；Prompt优化；开源模型替代；用量监控 |
| 指标 | AI成本/收入 < 30% |

#### 变现能力弱

| 项目 | 说明 |
|------|------|

| 指标 | 付费转化率 > 3% |

#### 竞争加剧

|------|------|

| 影响 | 高 |
| 概率 | 中 |
| 应对 | 差异化深耕；用户粘性建设；生态壁垒 |
| 指标 | NPS > 40 |

### 4. 运营风险

#### 团队扩展管理

| 项目 | 说明 |
|------|------|

## 风险应对流程

```
风险识别 → 评估影响/概率 → 制定应对 → 执行监控 → 复盘优化

日常监控：
├── 周报：风险指标追踪
├── 双周：风险应对复盘
└── 月度：风险矩阵更新

触发升级：
├── 黄色：团队内部解决
├── 橙色：需管理层介入
└── 红色：需全员动员解决
```

## 核心指标预警

| 指标 | 绿 | 黄 | 红 | 应对 |
|------|----|----|----|------|
| DAU/MAU | > 40% | 30-40% | < 30% | 增加触达、优化留存 |
| 任务完成率 | > 50% | 40-50% | < 40% | 调整难度、增加辅导 |
| 付费转化率 | > 5% | 3-5% | < 3% | 优化付费墙、价值展示 |
| AI成本/收入 | < 20% | 20-30% | > 30% | 降本、优化用量 |
| 次月留存 | > 35% | 30-35% | < 30% | 社群、情感、增强粘性 |
| 核心员工留存 | > 90% | 85-90% | < 85% | 文化、激励、沟通 |
