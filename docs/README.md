## 项目背景

"AI人生成长伴侣"是一个AI赋能的全生命周期成长生态平台，帮助用户在职业、认知、情感、健康等多维度实现自我迭代。

**核心差异化**：元路径概念（技能路径与人生阶段挂钩）+ 游戏化任务 + AI导师陪伴

**三阶段规划**：MVP (0-6月) → 扩展 (7-12月) → 生态 (13-24月)
